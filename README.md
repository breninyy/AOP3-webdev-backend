# 🔄 Processamento Assíncrono com XMLHttpRequest

Este projeto demonstra a implementação de **requisições assíncronas** utilizando o objeto `XMLHttpRequest`, permitindo a comunicação com APIs REST e a atualização dinâmica da interface sem recarregar a página.

---

## 🚀 Objetivo

Apresentar o funcionamento do processamento assíncrono em aplicações web, desde a ação do usuário até o tratamento da resposta e atualização do DOM.

---

## ⚙️ Funcionamento

1. O usuário realiza uma ação (ex: clique em botão)  
2. O JavaScript cria um objeto `XMLHttpRequest`  
3. A requisição é configurada com método, URL e modo assíncrono  
4. A requisição é enviada ao servidor  
5. A API processa e retorna os dados (JSON)  
6. O navegador recebe a resposta  
7. Os dados são tratados e exibidos na interface  

---

## 🧪 Tecnologias

- HTML  
- JavaScript  
- XMLHttpRequest
